🚚 FleetFlow — Smart Fleet Management System

📌 Overview

FleetFlow is a modern Fleet Management Dashboard designed to manage vehicles, trips, maintenance, expenses, performance analytics, and operational data in a clean and intuitive interface.

This project focuses on:
Real-world fleet workflow simulation
Clean dashboard UI with consistent theme
Modular scalable architecture
Modern React + Tailwind UI practices

✨ Features
🧭 Dashboard
Fleet overview layout
KPI cards
Trip summary table
Quick actions buttons
Unified layout & responsive design

New Vehicle Registration Form
