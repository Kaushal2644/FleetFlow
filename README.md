🚚 FleetFlow — Smart Fleet Management System

A modern Fleet Management Dashboard built with React + Tailwind CSS for managing vehicles, trips, maintenance, expenses, performance analytics, and operational workflows.

📌 Overview

FleetFlow is a real-world inspired fleet workflow simulation dashboard designed with a clean and consistent UI.

The system focuses on:

🚛 Real-world Fleet Workflow Simulation

🎨 Clean Dashboard UI with consistent theme

🧩 Modular & scalable architecture

⚡ Modern React + Tailwind practices
